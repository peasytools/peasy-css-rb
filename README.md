# peasy-css

[![Gem Version](https://badge.fury.io/rb/peasy-css.svg)](https://rubygems.org/gems/peasy-css)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

CSS code generator for Ruby — create linear/radial gradients, box/text shadows, border-radius, flexbox layouts, and CSS grid. Zero dependencies, outputs copy-paste CSS.

Built from [Peasycss](https://peasycss.com), a free online toolkit.

## Install

```bash
gem install peasy-css
```

## Quick Start

```ruby
require "peasy_css"

# Check version
puts PeasyCSS::VERSION  # => "0.1.1"
```
